# mollyweb
Clone of the Official Molly Eskam Web Portal - https://www.mollyeskam.net
An Apache2 installation on Debian is preferred.

# installation
1. Download the release file.
# wget
2. Run dpkg
# sudo dpkg -i mollyweb-v1.1-all.deb
3. Wait for finish.
